FPL Data Analysis

Fantasy Premier League (FPL) is an online game where you create a draft dream team and compete against other players. This game follows real life matches where the players you pick score points based on their individual performance, as well as team performance. With a limited budget, you have to make decisions on which players you want to include in your team. The game requires you to pick two goalkeepers, 5 defenders, 5 midfielders and 3 forwards.
FPL release the week by week data after each gameweek is played, this data is scrapped of their website and cleaned for data analysis.
A secondary statistics database called FBref is also scrapped to obtain more detailed data on teams and players. The aim of this project is to make informed decisions on which players to transfer in and out for specific gameweeks.

Python was used in this project to scrape data and perform data analysis.

Image Classification Research Project

My research was titled:
Using Deep Learning to Investigate Food Preferences in Small UK Garden Birds During Winter.

In this project I aimed partially automate the data collection and analysis as part of my thesis. I created an Image Classification model using Keras and TensorFlow to automatically classify species of birds based on images.
The model was trained on 30,000 images that were prelabelled on iNaturalist (A citizen science database). I improved my image classification model by Microsoft's MegaDetector which is an object detection model. Using MegaDetector I extrated the bounding-boxes surrounding the birds identified in the training data and cropped them. Doing this allowed me to improve the accuracy of my image classification model, as the model would learn the specificitys of the birds rather than the background in the bird images.
I then used my newly trained model on my data (Images I collected) to then classify the species of birds in my research field.
Applying MegaDetector and my image classification model, I automated the filtering and classification of 41,371 images collected during my research project. This process took roughly 5 hours to complete on my home PC, in comparison to 40 hours it would take to manually classify each image.
Whilst no new findings were discovered in this thesis, the application of Computer Vision in Ecology could significantly improve data processing and decision making in Ecology.

I'm currently going through this project again, fixing bugs and making the code universal so that anyone can pull the repository and build their own image classification model.
